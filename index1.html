HTML

HTML - HyperText Markup Language is the standart most used programming language to write a web page. HTML documents include text content (what you see) and markup (how it looks like).

Tags

HTML documents are made of HTML elements. Elements are created from tags.HTML tags are special marks that specify the format of content. Most of tags come in pairs, means you have an opening and closing tag. Some of them are void and need no closing tag.

Part2: HTML, CSS and DOM

CSS & DOM

Cascading Style Sheets (CSS) is a design language for electronic documents. With CSS you can determine for example layouts, colours and typography of your page. Together with HTML and DOM it is the key language of a World Wide Web.

Document Object Model (DOM) is organized in a tree-like structure. Every HTML document contains three main nodes, wich are: elements, attributes and text. In praxis a browser turns HTML tags into elements that form a tree. It lookes like a lot of boxes in boxes in boxes. This video gives you a very good explanation.

The Box Model
The Box Model consist of four borders, which have separate properties. These borders are:
  • content
  • padding
  • border
  • margin
There are two possibilities to calculate the box size, in pixels or in percentages. If the box-size is calculated in percentages, the size of the box will changing depending of how big the screen is or encompassing the box is. When the box-size is calculated in pixels, the box will have exactly same size no matter what happens.

To adapt the width of padding and border automaticly you can use the next CSS option:
*{ box-sizing: border-box; }

To be sure that all sites can display this code, you need to add some browserspecific prefixes. So it looks this way:
*{
-webkit-box-sizing: border-box;
-moz-box-sizing: border-box;
-ms-box-sizing: border-box;
box-sizing: border-box;
}
